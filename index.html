HTML

  • The Head element contains the "brain" of the webpage
  • The Body element contains the visibile content of the webpage
  • Use H1-h6 for headers
  • li for bullet points
  • script is linked to body
  • style is linked to head

CSS

  • must use "link" to link style to html file above title but below meta
  • href for source of external resource
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • class selectors are a must

Git

  • git status: checks current branch
  • git checkout -b branch-name: creates a new branch and switches to it

    • Commands for git bash

  • Ls = list items in current directory
  • Cd = navigates between directories in path
  • Pwd = displays current directory
  • Cd .. = goes back one step
  • Mkdir = make directory
  • Touch = creates file inside directory
  • Start = open
  • Code . = open vs code
  • git add -a = add all changes
  • git commit -m "hi" = commit changes with a message
  • git pull origin main = check local branch is up to date
  • git push origin filename = push change to github repo

JavaScript

  • variable is a named container for data
  • strings need quotation marks
  • lines of code in java end with a semicolon;
  • an array is a single variable used to hold a group of data usually used to hold related data
  • numbers and boolean values do not require quotation marks
  • arrays are zero-indexed and sequential (start at 0)
  • functions are reusable blocks of code that contain instructions for specific tasks
  • use curly brackets {} to contain code for a task in a function
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.(top to bottom)
